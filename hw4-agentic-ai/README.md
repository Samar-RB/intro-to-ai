# Homework 4 – Agentic AI (LLM Agents)

## Overview
This project implements an agent-based AI system using large language models (LLMs).

The system consists of multiple agents that solve a selected AI problem, compare solutions, and analyze differences using an "LLM as a judge" approach.

## Topics
- Agentic AI
- LLM-based systems
- Multi-agent architecture
- LLM as a judge
- Prompt engineering

## Features
- LLM agent that uses implemented algorithms
- LLM agent that solves the problem independently
- LLM judge agent that compares results and explains differences
- Integration with LangGraph framework

## Technologies
- Python
- LLMs (Gemini / Ollama / others)
- LangGraph
- Jupyter Notebook

## Files
- `AgenticAI.ipynb` – main implementation

## Skills Demonstrated
- Building multi-agent AI systems
- Integrating LLMs with classical AI methods
- Designing agent workflows
- Comparing AI-generated solutions
