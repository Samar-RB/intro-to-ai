# Homework 3 – Planning with PDDL

## Overview
This assignment focuses on solving a planning problem using PDDL (Planning Domain Definition Language).

The task involves modeling a real-world scenario (community garden) by defining actions, preconditions, and effects, and generating valid plans using a domain-independent planner.

## Topics
- Automated planning
- PDDL modeling
- Domain and problem definition
- Preconditions and effects
- Planning systems

## Files
- `community_garden.pddl` – domain definition
- `community_garden_problem_v*.pddl` – problem instances
- `community_garden_problem_v*_solution.txt` – generated plans

## Skills Demonstrated
- Modeling real-world problems as planning tasks
- Defining actions with preconditions and effects
- Working with PDDL
- Using automated planners
- Logical reasoning in AI
