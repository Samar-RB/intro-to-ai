# Homework 5 – Reinforcement Learning (Q-Learning & Value Iteration)

## Overview
This assignment focuses on implementing reinforcement learning algorithms to solve Markov Decision Process (MDP) problems.

The project includes Q-learning for the Cliff Walking environment and Value Iteration for the Gambler problem.

## Topics
- Reinforcement Learning
- Markov Decision Processes (MDP)
- Q-Learning
- Value Iteration
- Policy optimization

## Files
- `q_learning.py` – Q-learning implementation
- `value_iteration.py` – Value Iteration algorithm
- `cliff_walking_experiment.py` – RL environment experiment
- `gamblers_experiment.py` – gambler problem experiment
- `plotting.py` – visualization of results
- `common.py` – shared utilities

## Skills Demonstrated
- Reinforcement learning implementation
- Bellman equation application
- Policy optimization
- Working with environments (Gymnasium)
- Experimental evaluation and visualization
